## Purpose

This folder contains the engineering knowledge system for GhostMind.

GhostMind is treated as a governed private AI runtime platform, not merely a model, chatbot, NFT layer, or UI layer.

The core platform foundation is:

```text
Governance + Replay + Provenance + Security + Observability
```

Everything else, including RAG, agents, HUD, analytics, Digital Twin and NFT layers, is built on top of that foundation.
