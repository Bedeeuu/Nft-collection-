# RAG Pipeline

Canonical document lifecycle:

1. Upload
2. Quarantine
3. Scan
4. Provenance record
5. Integrity hash
6. Classification
7. Governance check
8. Chunking
9. Embedding
10. Vector storage
11. Retrieval permission check
12. Context assembly
13. Replay record

Rule: documents must pass quarantine and provenance validation before embedding.
